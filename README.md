# Card-Jitsu Snow Sprite Unpacker

Unpacks the sprites from Card-Jitsu Snow's TPK files.

## Requirements

- Python 3
  - Pillow
  - termcolor
- ImageMagick

## Usage

```
unpacker.py [-h] [--output OUTPUT] [--keep-jxr] [--extract-frames] [--create-gif] input
```

Options are self-explaining.
